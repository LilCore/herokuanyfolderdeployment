# Script para hacer despliegue de un directorio especifico en Heroku

Seguir video: https://www.youtube.com/watch?v=-j5gM2lCDus

1- Se debe añadir la variable PROJECT_PATH con el nombre del directorio que queremos desplegar en Heroku
2- Se debe añadir el link de este repositorio en los buildpacks de Heroku
